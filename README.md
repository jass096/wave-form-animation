# Wave Form Animation

This is a simple wave form animation project created using HTML and CSS.
It is a beginner-friendly project focused on animations and basic styling.

## Technologies Used
- HTML
- CSS
- JAVASCRIPT

## Description
The project shows a wave-like animation effect using CSS animations.
It helps beginners understand how animations and keyframes work.

## Features
- Smooth wave animation
- Clean and simple design
- Easy to understand code

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in any web browser
